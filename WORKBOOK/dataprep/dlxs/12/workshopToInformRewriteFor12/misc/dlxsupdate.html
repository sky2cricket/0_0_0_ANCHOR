DLXS: Recent Changes and Work Underway/Scheduled:

The most recent release of DLXS Middleware can be seen in the latest DLXS Release Notes: http://dev-linux.umdl.umich.edu/d/dlxs/docs/program/releasenotes.xml

  • Main additions to Release 12 include:
    • XML / XSLT delivery (BibClass to be finished in Release 13)
    • Cookie to hold session
    • Dynamic browse (Image Class and Text Class)
    • Pageviewer support for continuous tone images
    • Item level authorization (Image Class and Text Class)
    • Unicode / UTF-8 encoded text, indexable by XPATu
    • old home.tpl replaced by collection-specific index.html
    • language localization and collection-specific user interface strings
    • prototype for newspaper viewing
    • wordwheel exclusively UTF-8
