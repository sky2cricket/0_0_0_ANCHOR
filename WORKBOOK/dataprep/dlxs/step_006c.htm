modsICsearchOaiMusic finds or calculates the persistant url (purl) value 
	and uses the purl value with the input mods/ files to create output files
	for ic/, modsbib/, newmods/, preservation/, and search/.

First modsICsearchOaiMusic looks for the xml base value from the mods file in linkfile.

	The line for 0018_000054_000232_0000 xml base is:
	0018_000054_000232_0000  http://idserver.utk.edu/?id=200800000001660
	view short version of linkfile

If modsICsearchOaiMusic does not find the xml base value in linkfile, the locserve 
	database will generate a new purl value that is added to linkfile.  
Then output files are generated.
