6. Run the modsTC program that is in /d1/workarea/dlxs/text/tel/
	command> perl modsTC
	screen capture

modsTC finds or calculates the persistant url (purl) value 
	and uses the purl value with the input ocr/mods/ files to create output files
	for modsbib/, preservation/ocr/, search/ and tei/ocr/.

First modsTC looks for the xml base value from the mods file in linkfile.

	The line for 0022_000060_000204_0000 xml base is:
	0022_000060_000204_0000  http://idserver.utk.edu/?id=200800000001402
	view short version of linkfile

If modsTC does not find the xml base value in linkfile, the locserve 
	database will generate a new purl value that is added to linkfile.  
Then output files are generated.
