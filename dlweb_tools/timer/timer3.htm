timer3.htm set for 300,000 milliseconds (5 min)
adapted from: http://www.devguru.com/technologies/javascript/10904.asp
Text of the form and the function: 

<form name="myForm" action="" method="POST">
<input name="myClock" type="Text" size="40">
</form>
<script language=javascript>
var myseconds=300000; //5 minutes
self.setInterval('clock()', myseconds)
function clock() {
    time=new Date()
    document.myForm.myClock.value=time
    alert("timer3 time:\n"+time);
}
</script>
