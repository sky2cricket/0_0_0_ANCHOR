CodeGenerator by Cricket Haygood Deane
This is a project I am working on right now. This is a team effort. I provided a framework for cgi-form input, a database to support the project, and integrated code from artists with the cgi code generation. The goal is to make a web banner from CGI input.

I wrote most of this in perl, with a little javascript. But because this is cross platform development [unix, windows98, windowsNT], I wrote some htx/idc code to retrieve and display data from a SQL6.5 database.

There are traps for bad text strings, traps for non-authorized users, traps for a banner less than 8 months from the previous banner.

You can try out the kit with the username "guest".

The Code Generator takes up a great deal of page space. Depending on your browser and the size of your screen, you may wish to generate a new window.
