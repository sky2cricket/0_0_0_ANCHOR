C, C++ Programs by Cricket Haygood Deane
C language

The doUtility.c program is meant to take some of the burden out of running standard utilities on multiple files. This program generates a shell script that contains command line arguments for an entire batch of files. Run this program to generate the script, then run the script to execute the unix uitility. The example in this program is the unix diff utility, but any command line utility may be substituted into this general approach.

The safeRemoveByAge.c program removes files matching designated file extension(s) from the current directory, which must be defined by full path. There are safety features to prevent this program from being executed in the "wrong" directory.

The ascii2integer.cprogram is a short utility to determine ascii values for keystrokes.

C++ Language

The matrixPrint.cprogram demonstrates entering data into a matrix with C++.

The GreatestCommonDenomenator.cprogram demonstrates solving Euclid's Algorithm for integers entered from the keyboard.
