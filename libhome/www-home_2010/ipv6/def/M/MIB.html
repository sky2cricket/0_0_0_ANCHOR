Management Information Base (MIB)

The set of parameters of SNMP management station can query or set in the SNMP agent of a network device (e.g. router). Standard, minimal MIBs have been defined, and vendors often have Private enterprise MIBs. In theory, any SNMP manager can talk to any SNMP agent with a properly defined MIB. See also: client-server mode., Simple Network Management Protocol. [Source: BIG-LAN] (From RFC1983.)
