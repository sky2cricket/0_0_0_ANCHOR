Message Digest (MD-2, MD-4, MD-5)

Message digests are algorithmic operations, generally performed on text, which produce a unique signature for that text. MD-2, described in RFC1319, MD-4, described in RFC1320, and MD-5, described in RFC1321, all produce a 128-bit signature. They differ in their operating speed and resistance to crypto-analytic attack. Generally, one must be traded off for the other. (From RFC1983.)
