UNIX-to-UNIX CoPy (UUCP)

This was initially a program run under the UNIX operating system that allowed one UNIX system to send files to another UNIX system vial dial-up phone lines. Today, the term is more commonly used to describe the large international network which uses the UUCP protocol to pass news and electronic mail. See also: Electronic Mail, Usenet. (From RFC1983.)
