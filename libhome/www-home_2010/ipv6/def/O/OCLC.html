Online Computer Library Catalog (OCLC)

OCLC is a nonprofit membership organization offering computer-based services to libraries, educational organizations, and their users. The OCLC library information network connects more than 10,000 libraries worlwide. Libraries use the OCLC System for cataloging, interlibrary loan, and reference searching. [Source: OCLC] (From RFC1983.)
