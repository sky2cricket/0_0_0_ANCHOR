protocol

A formal description of message formats and the rules two computers must follow to exchange those messages. Protocols can describe low-level details of machine-to-machine interfaces (e.g., the order in which bits and bytes are sent across a wire) or high-level exchanges betweeen allocation programs (e.g., the way in which two programs transfer a file across the Internet). [Source: MALAMUD] (From RFC1983.)
