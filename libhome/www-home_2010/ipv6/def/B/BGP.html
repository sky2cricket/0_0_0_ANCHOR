Border Gateway Protocol (BGP)

The Border Gateway Protocol is an exterior gateway protocol defined in RFC1771. Its design is based on experience gained with EGP, as defined in RFC904 and EGP usage in the NSFNET Backbone, as described in RFC1902 and RFC1903. See also: Exterior Gateway Protocol. (From RFC1983.)

Border Gateway Protocol. The Internet is partitioned into different namespaces, sometimes referred to as different autonomous systems. The Border Gateway Protocol governs how routing is handled when crossing from one autonomous system to another. The current version of the protocol is BGP-4.
