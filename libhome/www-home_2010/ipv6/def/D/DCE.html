Distributed Computing Environment (DCE)

An architecture of standard programming interfaces, conventions, and server functionalities (e.g. naming, distributed file system, remote procedure call) for distributing applications transparently across networks of heterogeneous computers. Promoted and controlled by the Open Software Foundation (OSF), a consortium led by Digital, IBM, and Hewlett Packard. [Source: RFC1208] (From RFC1983.)
