Defense Data Network Network Information Center (DDN NIC)

Previously called "The NIC", the DDN NIC's primary responsibility was the assignment of Internet network addresses and Autonomous System numbers, the administration of the root domain, and providing information and support services to the Internet for the DDN. Since the creation of the InterNIC, the DDN NIC performs thses functions only for the DDN. See also: Autonomous System, network addresss, Internet Registry, InterNIC, Network Information Center, Request for Comments. (From RFC1983.)
