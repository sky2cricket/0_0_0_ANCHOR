Appletalk

A networking protocol developed by Apple Computer for a communication between Apple Computer products and other computers. This protocol is independent of the network layer on which it is run. Current implementations exist for Localtalk, a 235kb/s local area network; and Ethertalk, a 10Mb/s local area network. [Source: NNSC] (From RFC1983.)
