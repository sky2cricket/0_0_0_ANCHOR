How to use Netscape 4 editor.

1.  click on File icon (upper left corner).
     This will cause a menu to appear.
 2.  Select (by clicking) the Edit Page item on menu.
  3.  Wait for a new screen to appear, titled:
         "Netscape Composer"
        This screen will hold your original file.
  4.   Make your editing changes (some possibilities are described below).
  5.   Click on the  File icon in the upper left corner of the "Netscape Composer" Screen.
 6.  Select (by clicking)  the Save  option.   A new screen will come up asking if you
      wish to OVERWRITE  the original file.
     a.  If the answer is yes,  click on OK.
     b.  If the answer is no, click on cancel.
 7.  If you with to create a NEW file instead of overwriting your original file:
      a.  Click on the File icon in the upper left corner of the "Netscape Composer" Screen.
      b.  Select (by clicking) the Save as... option.  A new screen will come up asking you
           for a file name.  Type in the name you wish in the space provided.
      c.  Click on OK.
