BROWSER DETECTION --- Make your web page display compatable with the
current client.  Use JavaScript to detect the browser and then use the detected
values to select code appropriate to the current browser. (See the next example 
---OPEN A SPECIAL WINDOW--- for implementation of browser detection to control 
JavaScript code.)
<p>
The table below is generated by a series of JavaScript document.write commands.
The values that appear in this table (middle column below) will depend upon 
the browser you are using.
<p>
<SCRIPT language="JavaScript">
document.write("<H4><P>");
document.write("<TABLE BORDER=2 CELLPADDING=5>");
document.write("<tr><td><h4>Browser</td>");
document.write("    <td><h4>Value</td>");
document.write("    <td><h4>JavaScript Code</td></tr>");
document.write("<tr><td><h4>Name</td>");
document.write("    <td><h4>"+navigator.appName+"</td>");
document.write("    <td><h4>  navigator.appName </td></tr>");
document.write("<tr><td><h4>Version</td>");

<A NAME="SPECIAL">
<hr><hr><hr>
OPEN A SPECIAL WINDOW ---  You can use JavaScript to open a new
specialized window with size and properties specified by you.  (size of window,
tool bars, location of window relative to current window, etc. ) 
<p>
With Netscape 3 and higher, you can keep the special window in the front of
the display (focus method) or cause it to be hidden behind the current window
(blur method).
<p>
This example also shows why Browser Detection should be used with your JavaScript
code. Here, if the browser version is detected to be less than 3, the focus and
blur methods are not available and will not applied.  If you do not take care of 
this detail, users with older browsers will be penalized by errors produced by 
your JavaScript code.
<p>
Hiding the special window puts it out of view.  Closing the special window frees 
all resources associated with it.  When you write code to open special windows,
you should always provide a close option because you do not know the memory 
resources of your client. 
<p>
<SCRIPT language="JavaScript">
var SWopts = "toolbar=0,location=0,directories=0,status=0,menubar=0,scrollbars=0,copyhistory=0,resizable=0,width=200,height=400"
var bVer=parseInt(navigator.appVersion);
function DisplaySpecialWindow(href_name,Nchoice){
  SWname = "SpecialWindow";
  SWwindow = window.open(href_name,SWname,SWopts);
  if(bVer > 2) { 
	if(Nchoice==1) SWwindow.focus(); 
        if(Nchoice==2) SWwindow.blur();
	}
  if(Nchoice==0)SWwindow.close();
}
</script>
