error 80004005
Can also be caused by needing to have a Response.Buffer for your asp page.
To get a Response.Buffer use this code: 
< % @ Language=JAVASCRIPT % >
< % Response.Buffer = true % > 
__________OR__________________
< % @ Language=VBSCRIPT % >
< % response.Buffer = True % >
