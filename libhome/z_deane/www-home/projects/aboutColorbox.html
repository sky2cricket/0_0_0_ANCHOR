Colorbox by Cricket Haygood Deane
Have you ever wanted to find that "perfect color" for the background of your HTML page?

Try my Colorbox program. It's as easy as point and click.

This is a perl-cgi program I wrote to entertain my niece Ginny over the Christmas break.

The functions "Random Background" and "Random Text" use a random number generator based on the current time value to dynamically produce a new set of 3 hexadecimal values when the button is pushed.

There is a filter on text input to prevent the entry of malicious strings in the text input box. For example, try entering the input string "UVWXYZ". It will be reset to "000000".

Enjoy.
