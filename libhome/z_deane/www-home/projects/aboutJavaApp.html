Java Application by Cricket Haygood Deane
This code was written as part of the Performance Monitoring Project of Innovative Computing Labs, in 1999. It is designed to run dynamically off datat generated by the performance monitor. The demonstation code here uses some performance data "captured" during a test run.

This is really a set of programs in different languages (java, c, HTML) that work together.

If you just want to see the PerfometerApplet in action, click on PerfometerApplet Only.

If you prefer to see the full demonstration page with all details and all code, click on PerfometerApplet Full Demo.
