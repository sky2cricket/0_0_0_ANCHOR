How to use Netscape 4 editor.

BEFORE YOU START...

First,  answer this question:
            From what directory did I  type in the command  netscape4?????
 
The files you create will be automatically deposited into this directory,
           unless you specify otherwise.

 For example, if the  file that bubba wishes to edit is in ~bubba/www-home/MYTEST.html
           and Bubba  call s up netscape4  from his TOP directory,
           the  Save command will make a file  ~bubba/MYTEST.html in the TOP directory.

  And then Bubba will spend 30 minutes searching for the modified MYTEST.html file
           in his www-home directory until Dr. Straight comes and rescues him.

   So either modify your filename to get it into the correct directory or issue the
           Netscape4 command from the correct directory.


NOW YOU ARE READY TO START EDITING YOUR .html FILE

0.  Goto the file you wish to edit in Netscape4 by typing in the
      URL in the  "Location"  option on the top of your netscape screen.
1.  Once your page has come up,  click on File icon (upper left corner).
    This will cause a menu to appear.
 2.  Select (by clicking) the Edit Page item on menu.
  3.  Wait for a new screen to appear, titled:
         "Netscape Composer"
        This screen will hold your original file.
  4.   Make your editing changes (some possibilities are described below).
  5.   Click on the  File icon in the upper left corner of the "Netscape Composer" Screen.
 6.  Select (by clicking)  the Save  option.   A new screen will come up asking if you
      wish to OVERWRITE  the original file.
     a.  If the answer is yes,  click on OK.
     b.  If the answer is no, click on cancel.
 7.  If you with to create a NEW file instead of overwriting your original file:
      a.  Click on the File icon in the upper left corner of the "Netscape Composer" Screen.
      b.  Select (by clicking) the Save as... option.  A new screen will come up asking you
           for a file name.  Type in the name you wish in the space provided.
      c.  Click on OK.

SOME POSSIBILITIES FOR EDITING YOUR .html FILE.

  1.  You can check your spelling by using the  "Tools" option on the
        top line of your "Netscape Composer" window.
           a.  Click on Tools.  A menu will appear.
           b.   Click on Spelling...   Several things will happen.
                   1.  All of your non-standard or misspelled words will be underlined in red.
                    2.   A small screen will appear with some options.  (Unfortunately,  this
                          screen covers up your text and I don't know what to do about that.)
                    3.  Suggestions for spelling corrections are given.

   2.  You can revise your file.   This is like  normal editor programs.  You put your
         cursor where you want to type and start typing.  To back up, use back space.
         To delete, go to the delete button.  If you are used to windows95 typing, you
          will be happy.  If you are used to typing with vi, you will go crazy.  Oh well.
          :wq
 
     3.   You can change the  SIZE and STYLE of your type by going to the
            "Font" option on the top line of your "Netscape Composer" window.
            a.  Click on Format.   A menu will appear.
            b.  Options include:  Font, Size, Style, Color, Remove all styles,
                  Heading, Paragraph, List, Align, Increase Indent, Decrease Indent,
                  Character Properties, Page Colors and Properties.
             c.   Click on ANY of these choices and
                          1.  A new select menu will appear or...
                           2.  A screen with options will appear....
                           3.  The selected change will be directly applied to your page.
                                 (i.e. Increase Indent automatically increases paragraph indentation.)

      4.  Create HYPERTEXT links!
             a.  Click on the  "Insert" option on the top line of your "Netscape Composer" window.
                  A menu will appear.
             b.   Select  "Link" by clicking on it.  A small screen will appear.  It will have
                   2 places for you to type.
                            1.  Link Source. Type the TEXT that you will click on.
                             2.  Link To.   Type the URL of the target (including http:// )
                              3.  Sorry, I haven't figured out how to do this with icons.

      5.   Make a TABLE.    I just clicked on the "Table" option on the Second Line
            of the "Netscape Composer" Screen.  It made this thing you see below. **** If you are NOT in Netscape4, the empty table will not appear. *****
 
             So Now I will click on it again.   It makes another empty table thing.
 
So I put the cursor inside the empty table and type.   It puts the words inside the table.   Amazing!

            If you add a NEW table to a file, you must save the file to a new filename.  Netscape4
            will not add tables to an existing file.  But once the new file containing the table has
            been created, you can revise the new file and even revise the words in the table in
            the new file.
