Hypertext Transfer Protocol (HTTP)

The protocol used by WWW to transfer HTML files. A formal standard is still under development in the IETF. See also: hyperlink, hypertext, Hypertext Markup Language, World Wide Web. (From RFC1983.)

Hypertext Transfer Protocol. This is a set of rules governing the addresses, structure, and format of web pages.

More information at:

RFC2109: HTTP STATE Management Mechanism, February 1997.

RFC2070: Internationalization of the Hypertext Markup Language, January 1997.

RFC2069: An Extension to HTTP: Digest Access Authentication, January 1997.

RFC2068: Hypertext Transfer Protocol, HTTP/1.1, January 1997.

RFC1945: Hypertext Transfer Protocol, HTTP/1.0, May 1996.

RFC1942: HTML Tables, May 1996.
