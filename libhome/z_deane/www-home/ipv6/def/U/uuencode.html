uuencode

A program which reversibly converts a binary file in ASCII. It is used to send binary files via email, which generally does not allow (or garbles) the transmission of binary informaiton. The original binary can be restored wtih uudecode. The encoding process generally creates an ASCII file larger thatn the original binary, so compressing the binary before running uuencode is highly recommended. (From RFC1983.)
