Acceptable Use Policy (AUP)

Many transit networks have policies which restrict the use to which the network may be put. For example, some networks may only be used for non-commercial purposes. Some AUPs limit the type of material which can be made available to the public (e.g., pornographic material). Enforcement of AUPs varies with the network. See also: netiquette. (From RFC1983.)
