Simple Network Management Protocol (SNMP)

The Internet standard protocol developed to manage nodes on an IP network. The first version is defined in RFC1157, (STD 15). SNMPv2 (version 2) is defined in too many RFCs to list. It is currently possible to manage wiring hubs, toasters, jukeboxes, etc. See also: Management Information Base. (From RFC1983.)
