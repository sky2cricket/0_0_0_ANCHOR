token ring

A token ring is a type of LAN with nodes wired into a ring. Each node constantly passes a control essage (token) on to the next; whichever node has the token can send a message. Often, "Token Ring" is used to refer to the IEEE 802.5 token ring standard, which is the most common type of token ring. See also: 802.x, Local Area Network. (From RFC1983.)
