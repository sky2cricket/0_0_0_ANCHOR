Reverse Address Resolution Protocol (RARP)

A protocol, defined in RFC903, which provides the revers function of ARP. RARP maps a hardware (MAC) address to an internet addresss. It is used primarily by diskless noeds when they first initialize to find their internet address. See also: Address Resolution Protocol, BOOTP, internet address, MAC address. (From RFC1983.)
