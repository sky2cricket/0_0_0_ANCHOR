How to DownLoad with LYNX text only browser:

0. From window "A" call up netscape

1. Find the URL (or image) you like with Netscape

2. In another window "B" cd to your www-home directory

3. In window "B", type lynx (which loads lynx browser)

4. Enter the lynx command G (means goto)

5. Type in the URL that you found in step 1 above.

6. Lynx will give you options, choose D (means download)

7. Make sure the filename you choose is unique

8. Type q (means quit) to exit lynx

9. Type chmod 755 * to change protections on your new file

10. Enjoy your new gif. I got most of the gifs on this page using this method.
