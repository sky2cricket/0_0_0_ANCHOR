These are the characters that everybody gets. I listed the ascii characters 060 and 062 because the angle brackets pose a particular obstacle when using HTML, since they denote tags and even using <pre> does not work. (See view source on netscape for this page and you will see what is going on.) For a comprehensive list of special characters (including unicode) the CS100 Guide to How to make your own Homepage.

Another handy character is the "enforce space" character. This overrules the HTML default of discarding whitespace. The character is written

&nbsp;
where NBSP stands for No Back SPace.

There is a math extension to HTML which some people get. To use the math extension, enclose the desired characters within math tags as shown below.

<math>desired characters</math>
