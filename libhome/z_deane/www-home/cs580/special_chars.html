Special Characters in HTML
by Cricket! Haygood Deane


Any ascii character can be represented in html using the format 

&#NNN;

where NNN is the ascii number (0 to 255). This also works for UNICODE characters, but my local system does not support UNICODE. I wish it did, because then we could read all those neat Japanese web pages!

For example, the symbol for copyright is © which I just created using &#NNN; with NNN=169. Look at view source to see how I did it.

So if you have a yearning to give a price in ¥ or ® a trademark or you just think §§§ is neat, use your &#NNN; option!

           To FORCE a single blank use &NBSP; with nspb as lower case. I used this method to indent this line. Look at view source to see how, since I can't figure out how to display this. It won't even display with pre tags.

The tables below are for reference. HTML represents unprintable characters with a little box or with a single blank. That's why the table is not too great for ascii values below 32. Since ASCII 32 is a single blank, that is how it comes out on the table.
