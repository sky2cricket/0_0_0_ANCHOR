Text Class Graphics Files

The following table describes those graphics files that are likely to be used by any instance of Text Class. All of the files below should be strored in the <dlxsroot>/t/text/graphics directory, and in each subsequent ../c/collection/graphics directory. Saving any alternative graphics by these names in individual graphics directories will result in variation of appearance within template-generated pages.

Image File name Description
aro-upw.gif Appears in all scrollable pages as a "back to top" button.
ww-dw-000000.gif Scroll down button on ww-full.tpl
ww-up-000000.gif Scroll up button on ww-full.tpl
sm-dwnarrow.gif Used to indicate an expanded menu in the help pages.
navarrow.gif Used to indicate an expandable menu in help files.
navdwnarrow.gif Used to indicate an expandable menu in the main navigation bar.
navdwnarrow-grey.gif Corresponding image to navdwnarrow.gif for highlighted (e.g., white) navigation tabs.
plug.gif Transparent spacer gif used throughout. hdr-bg1.gif, hdr-bg2.gif, and hdr-bg3.gif are also transparent gifs used as table background gifs for header displays.
onbullet.gif Used in a mulitple collection results list to help highlight the collection group whose results are currently being listed.
ret-arrow-generic-grey.gif Used in header.tpl and text.tpl as part of a link back to search results.
prevoff.gif Used in pageviewer.nav.tpl as the "up" stage of the previous page button.
prevon.gif Used in pageviewer.nav.tpl as the "mouseover" stage of the previous page button.
nextoff.gif Used in pageviewer.nav.tpl as the "up" stage of the next page button.
nexton.gif Used in pageviewer.nav.tpl as the "mouseover" stage of the next page button.
ret-arrow-generic.gif Used in pageviewer.nav.tpl as part of the link back to the results page.
scale.gif Used in pageviewer.nav.tpl as the scale menu label.
view.gif Used in pageviewer.nav.tpl as the page view menu label.
goto.gif Used in pageviewer.nav.tpl as the goto menu label.
