Web Server Overview

We give here, as an example, the DLPS environment.

  • Apache (version 1.3.27 as of this workshop date)
  • Virtual hosts for various collections with their own specific
    • DocumentRoot
    • ScriptAlias
    • DLXSROOT environment variable (see directory structure)
    • If no external Auth system, set the AUTHZD_COLL environment variable in the virtual host to contain a colon-separated list of authorized collections (Middleware expects this environment variable to be set)
