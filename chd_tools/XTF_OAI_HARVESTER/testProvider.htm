 4. Protocol Requests and Responses
   4.1. GetRecord ............................... needs metadataPrefix and identifier
   4.2. Identify ................................ no modifiers
   4.3. ListIdentifiers ......................... needs metadataPrefix, can have resumptionToken after first request
   4.4. ListMetadataFormats ..................... no modifiers
   4.5. ListRecords ............................. needs metadataPrefix, can have resumptionToken after first reuqest
   4.6. ListSets ................................ no modifiers
Set Base URL  
The value of otherOaiProvider can be any other oai provider that works. This can be used for easy comparison.
The value of yourPath corresponds to the tomcat5/webapps directory containing xft.
You can find this value by looking at the URL for one of your XTF online objects.
