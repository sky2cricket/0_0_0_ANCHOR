XPAT Overview

  • Originally OpenText’s pat50
  • As of the date of this workship, we are running XPAT version 5.3.2
  • DLPS acquired the source code, has made some changes and enhancements, plans more.
  • String based searching rather than word based
  • SGML/XML aware searching in the sense that one make very specific, structurally constrained searches for strings and regions
  • index and search Unicode (UTF-8) encoded texts
  • See XPAT details
