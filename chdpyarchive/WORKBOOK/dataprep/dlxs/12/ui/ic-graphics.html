Graphics Files Used in ImageClass

The following tables describes those default graphics for all instances of ImageClass. All of the files below should be strored in the <dlxsroot>/i/image/graphics directory. Saving any alternative graphics by these names in individual graphics directories will result in variation of appearance within template-generated pages.
