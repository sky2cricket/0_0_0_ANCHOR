MODS LABEL VALUE  
Title    
Three name and name attributes blocks allow for creation of up to 3 sets of mods:name tags.
The following Name Authorities are available:
  • LCNAF -- Complete Description at: http://authorities.loc.gov/
  • ULAN -- Complete Description at: http://www.getty.edu/research/conducting_research/vocabularies/ulan/
    • Primary Name Type IS THE PRIMARY NAME ASSOCIATED WITH THE CREATION OF THIS RESOURCE KNOWN?
         personal  If you select personal, you are required to fill in a Last Name/Corporate Name below. Name Parts (First name, Middle name, and Vital Dates) are recommended.
         corporate  If you select corporate, you are required to fill in a Last Name/Corporate Name below. Name Parts are not used.
         unknown  If you select unknown, the value unknown will be entered as the name.
         anonymous  If you select anonymous, the value anonymous will be entered as the name.     		 
    Name Authority      NONE  If you select NONE, no name authority will be added to the mods tag.
         LCNAF  If you select LCNAF and name type personal or corporate, LCNAF name authority will be added to the mods tag.
         ULAN If you select ULAN and name type personal, ULAN name authority will be added to the mods:name tag.
           Please note, name authority only applies to personal names or corporate names, not unknown or anonymous.
