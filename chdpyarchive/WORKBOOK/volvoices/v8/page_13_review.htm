Review Page

 
The Review Page:
  • provides a quick display of your work before saving the xml file to a local directory
  • shows what information has already been saved
  • is display only and causes no changes

    While creating a single MODS record, you may visit this page as often as you like to check your work.

    Check Required Fields.

  • On all input pages of the workbook, the input boxes and dropdowns with a yellow background are for REQUIRED fields.
  • On the Review Page, REQUIRED fields appear in rows with a yellow background.

    Check Content.

  • You are in the best position to know what should be entered into the workbook for the item you are describing.
  • If you find something amiss, this is the time to return to the input page, enter a correction, save, and review again.
