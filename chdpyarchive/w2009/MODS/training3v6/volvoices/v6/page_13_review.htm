This review page provides a quick display of your work before saving the xml file to a text document.
This review page shows what information has already been saved.
This review page is display only and causes no changes.
You may visit this page as often as you like to check your work.

Check Required Fields.

  • All Page Data. The rows with a yellow background are for REQUIRED fields. The rows with pale blue background are not required.
  • Page 6 Data. The 2 rows with a pink background on page 6 are conditionally REQUIRED. Either you have both of these or neither of these.

    Check Content.

  • You are in the best position to know what should be entered into the workbook.
  • If you find something amiss, this is the time to return to the input page, enter a correction, save, and review again.
