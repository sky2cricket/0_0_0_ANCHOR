ic setup using ic/0031_000050_000203_0000 as an example

Relevant files in this directory.
ic/*
done/ic/*
ic.vvz.unnorm.sgm
footer
vvzheader


When you run modsICsearchOai >nameYourRedirect.txt the input is
mods/* .................Workbook Mods for 0031_000050_000203 picked up by pickupTime 0031_000050_000203_0000.xml
The output is 
ic/* ...................file for 0031_000050_000203 in image class format 0031_000050_000203_0000


The ic files are used to create the unnormalized sgml file for image class.
View the header, the footer, and the very short example of ic.vvz.unnorm.sgm below.
vvzheader
footer
ic.vvz.unnorm.sgm(short)


The command to create the unnormalized sgml file for image class is shown below.
cat vvzHeader ic/* done/ic/* footer > ic.vvz.unnorm.sgm

Note:  The final filename (ic.vvz.unnorm.sgm) contains a designator for type/class, here vvz.  
       This designator should be all lower case letters.
       For example: if your type/class is UTKgsm, change it to utkgsm.

       This type/class designator is also used to name the directory ( here vvz )
       So for the vvz example we have the directory
		/d1/dlxs/prep/v/vvz
       This contains the files
		ic.vvz.norm.sgm
		ic.vvz.unnorm.sgm

These two files (ic.vvz.norm.sgm and ic.vvz.unnorm.sgm) will be replaced.

The new ic.vvz.unnorm.sgm file is copied to the /d1/dlxs/prep/v/vvz/ directory and overwrites the current version.
The current version of ic.vvz.norm.sgm is removed from the /d1/dlxs/prep/v/vvz/ directory.

In the next step (ic setup), the normalized sgm file (ic.vvz.norm.sgm) will be built by the program 
/d1/dlxs/bin/i/image/idb

The thumbs are copied to   /d1/dlxs/img/v/vvz/thumb/ directory.
The jpgs are copied to     /d1/dlxs/img/v/vvz/large/ directory.
There is a third directory /d1/dlxs/img/v/vvz/index/ that contains index information for images.

Summary of commands:

cat vvzHeader ic/* done/ic/* footer > ic.vvz.unnorm.sgm
cp ic.vvz.unnorm.sgm /d1/dlxs/prep/v/vvz/.
>>>causes check of overwrite, answer y:

>>>causes check of removal, answer y: 
>>>rm: remove regular file `/d1/dlxs/prep/v/vvz/ic.vvz.norm.sgm'? y
>>>this file is replaced in next step page 5d ic setup
