store1/area2/volvoices/vvImport/selected_directories

SCRIPTS TO RUN:

a.  command> perl check_incoming (makes sure xml file has all tiff children)
	output found in results/IncomingTest_timestamp.txt

b.  command> perl duplicates_noproc >nameYourRedirect.txt
    command> perl duplicates_noproc_reworked >nameYourRedirect.txt
    There are several versions of this script to find duplicates 
	(xml files already online)
   	output goes to screen unless you redirect

c.  command> perl onedirDerivatives (tests for xml having all children)
	Children needed can be tiffs, jpegs, minithumbs, thumbs, text.
	Exact children depend on directory. (e.g. 00 not need text)
	output goes to results/derivatives_timestamp.txt

d.  command> perl fieldtest (required xml tags populated)
	Requirements can change.
	output goes to results/FieldTesttimestamp.txt
	
AFTER FILE VALIDATION RUN PICKUPTIME
This will pickup scripts from 00,01,02,03,04,05 in your selected batch.

command> perl pickuptime > nameYourRedirect.txt

pickuptime will present you with a list of directories on
store1/area2/volvoices/vvImport to choose from.  Select the directory,
output will go to screen unless you redirect.

AFTER PICKUPTIME

Go to each directory and work the files.
